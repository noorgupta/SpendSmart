# SpendSmart

A SaaS Expense Tracker built using the MERN Stack.

## Features

- User Authentication
- Expense Tracking
- Income Tracking
- Budget Management
- Analytics Dashboard
- Recurring Transactions
- Report Export

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose
